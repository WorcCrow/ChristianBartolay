# Pro-Level Prompt for Generating an HTML Tutorial Page

Create a **single, fully functional HTML file with embedded CSS** that
demonstrates and introduces all the HTML topics listed below.

## Objective

The page should function as a **mini educational tutorial website** that
teaches basic HTML tags and structure. Each concept must be demonstrated
with **visible examples rendered on the page**.

## Output Requirements

1.  Produce **one HTML file only** with **internal CSS inside
    `<style>`**.
2.  The page must be **clean, modern, and visually presentable**.
3.  The layout should resemble a **tutorial or documentation page**.
4.  Organize the page into **multiple logical sections** grouping
    related topics.

## Section Requirements

Each section must include: - A **section title** - A **short description
explaining the concepts** - A line stating **which topic numbers from
the list are covered** - **Live examples showing the HTML elements in
action**

Example format:

**Section Title**\
Description: This section demonstrates basic text formatting elements.\
Topics Covered: 9, 10, 11, 12, 13

Example Output:\
(Rendered HTML example)

## Design Requirements

Use CSS to make the page visually appealing: - Centered content
container - Card-style sections - Soft background color - Clear
typography - Spacing and padding - Styled headings - Navigation or
anchor links between sections (optional)

## Structure Guidelines

The page should include: - Header with page title - Introduction
section - Multiple tutorial sections - Example demonstrations -
Horizontal separators where appropriate - A simple footer

## Demonstration Rules

-   Each HTML topic must be **clearly demonstrated visually**.
-   If possible, **show multiple tags in context** rather than isolated
    examples.
-   Keep examples **simple and beginner-friendly**.

## Code Quality

-   Use **proper indentation**
-   Use **clear comments**
-   Ensure the HTML is **valid and readable**
-   Use **semantic structure when possible**

## Goal

The final page should look like a **simple beginner-friendly HTML
tutorial webpage** that someone could use to learn the basics of HTML.
